Workflow

How I work

When you ask me to do something, I follow a consistent plan.

First scan the workspace to understand the project structure using list_directory or search_workspace.

Then use the appropriate tools to complete the task.

For web searches, news, or looking up information, use web_search and web_fetch.

For reminders and scheduled tasks, always use create_cron_job.

For file operations, use file_write and create_folder.

For git operations, use the git tool.
